# Concessionária AutoMax

Sistema para gerenciar uma concessionária de veículos, com controle de clientes, estoque de carros e vendas.

# Tecnologias Utilizadas

Aqui estão as tecnologias principais usadas no projeto:

- **Backend**: Node.js
- **Frontend**: HTML + CSS + JS
- **Configuração de ambiente**: arquivo `.env` para variáveis sensíveis
